A simple yet functional Streamlit chatbot application powered by OpenAI's GPT-3.5 model. This template demonstrates how to build a conversational AI interface with persistent chat history and streaming responses.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatbot-template.streamlit.app/)

## Features

- **Conversational Interface**: Interactive chat UI with message history
- **OpenAI GPT-3.5 Integration**: Powered by OpenAI's language model
- **Streaming Responses**: Real-time response generation for better UX
- **Session Persistence**: Chat history maintained throughout the session
- **Secure API Key Handling**: Password-protected API key input

## Prerequisites

- Python 3.7+
- OpenAI API key ([Get one here](https://platform.openai.com/account/api-keys))

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
